WWW

WWW is short for World Wide Web. The World Wide Web is a collection of html documents. Each docuent is made up of HTML or hypertext markup language, which is the basis for almost every web page on the internet, the glue.

Each paged is linked together via hyperlinks and this is what makes the web, web-like.

Invented in the 1990's and has around 30 Billion pages and growing exponentially.

HTML Basics

What is it made up off?

Text content - which is what you see on the page.
Markup - the visual.
Links to other docs such as images and videos.

HTML Elements

HTML is reasonably straightforward

the <"b"> tag makes words bold

I am going to learn to love HTML.

The <"em"> tag makes words italic

Attributes

<"a"> (anchor) tag is used for link

"href" is the attribute name for a link. This is where you provide the url which points to the page you want to visit on the web

<"img"> tag is used for images

"src" is the attribute name for an image.

This is an example of using links and images in HTML.

Inline vs Block

p tag is a block element, which gets wrapped with a HTML box and starts a new line.

Where as the "br" tag is an inline element, does NOT get wrapped with a HTML box,
but behaves similarly by creating a new line afer the tag.


The span tag is an inline element that just holds content which can be styled
The div tag is a block element that just holds content which can be styled as well.
